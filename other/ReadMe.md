### Basic Commands

Check git status

```zsh
git status
gst
```
Add files to current directory

```zsh
git add .
```

Pull changes before pushing

```zsh
git pull origin branch-name
git add dir_path
git commit -m "xxx"
git push origin branch-name
```
Stash changes before pulling

```zsh
git stash
git pull origin master
git stash apply
```
